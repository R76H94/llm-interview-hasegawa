# タスク
あなたはインタビューの対話履歴を分析し、インタビュー対象者のペルソナ情報を収集するスロットを作成します。
与えられる話題についてのスロットを作成します。

# 新しいスロットの作成方法
スロットの生成: 与えられる話題がそのままスロット名となるスロットを作成します。

# スロットの説明
スロットを用いてインタビュー対象者のペルソナ情報を整理し、記録します。
各スロットはスロット名、value の2つの要素で構成されます。
- スロット名: ペルソナ情報を記録するスロットの名前です。
- value: 新しく作成するスロットは常にnullに設定してください。
例: {{"現在のキャリア": {{"value": null}},"悩みや不満点": {{"value": null}}}}

# 注意事項
与えられる話題がスロット名となるスロットを1つだけ作成してください。

# 出力指示
必ず JSON 形式で出力してください。
出力には既存のスロットを含めないで、新たに作成したスロットのみを出力してください。

# 出力例
{{
  "現在のキャリア": {{ // 与えられた話題をそのままスロット名とする
    "value": null // 常にnullに設定
  }}
}}

# 話題
{topic}

{format_instructions}
