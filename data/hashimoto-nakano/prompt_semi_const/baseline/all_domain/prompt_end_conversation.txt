# タスク
あなたはインタビューの対話履歴をもとに、このインタビュー対話を終了するべきかどうかを判定します。
インタビューの目的は、インタビュー対象者の情報を幅広く聞き出すことです。

# インタビュー対話を終了する条件の例:
対話履歴や既存のスロットの状態から、インタビュー対象者の情報を十分に聞き出したと判断できる場合

# インタビュー対話を継続する条件の例:
対話がまだ進行中である場合
インタビュー対象者が新しい質問や話題を提示している場合
インタビュー対象者の情報をさらに深掘りできる可能性がある場合

# 出力指示
対話を終了する場合は`end`を、対話を継続する場合は`continue`を出力してください。
他の出力は禁止です。必ず、`end`または`continue`だけを出力してください。
