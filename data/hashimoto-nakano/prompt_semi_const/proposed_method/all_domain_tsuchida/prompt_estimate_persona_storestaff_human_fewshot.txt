# タスク
あなたは対話履歴と既存のスロットを分析し、インタビュー対象者が現時点でどのようなペルソナ属性を持っていると推定できるかを判断します。

# 前提
このインタビューの目的は、対象者の人物像の中で重要な側面を捉えることです。
ここでの「重要」とは、対話全体を通じて対象者を深く理解するために本質的な情報を意味します。
後に「重要」な側面の基準となる例を与えるのでそれに従ってペルソナ属性を取捨選択してください。
インタビューで聞き出したインタビュー対象者のペルソナ情報はスロットに格納されています。
ペルソナ属性とは、インタビュー対象者が持つ属性カテゴリ（例：「キャリア・職場」「悩みやストレス」「趣味」「家族」など）のことです。
推定ペルソナとは、インタビューの対話履歴と既存のスロットをもとに、「現時点でこのインタビュー対象者はどのようなペルソナ属性を持っている（または持っている可能性が高い）」を推定した結果です。

# 指示
あなたの最優先事項は、対象者の人物像の中で重要な側面を捉えることです。以下の手順に従って、ペルソナ属性を慎重に判断してください。
1. 情報の重要度判断:対話履歴と既存のスロットを分析し、各スロットの情報が、対象者の人物像を深く理解する上で、どれだけ重要かを判断します。
2. 候補情報の抽出:以下の条件を満たすスロットのみを「重要な情報」候補として抽出してください。条件を満たさないスロットは「重要でない情報」として、絶対にペルソナ属性としないでください。
  - スロットの値がnullでない。
  - スロットの値が「わかりません」「特にないです」「情報なし」など、対象者が明確に情報を持っていない・回答していないことを示す内容ではない。
3. 抽出した候補の重要度の判断:「重要な情報」候補として抽出したスロットについて、以下の観点から「重要」かどうかを評価してください。「重要」と判断したスロット情報のみをペルソナ属性として扱います。「重要」ではないスロットはペルソナ属性としないでください。
  - 対話の中で繰り返し言及されている、あるいは詳細に語られている
  - 他の情報と 関連して関連して人物像を深める役割を果たしている
  - 対象者自身が強調している／明示的な態度・意図を示している
  - 内容が曖昧でなくかなり具体的な回答をしている
  - 対話全体の文脈において人物理解に寄与していると判断できる
  - 対話履歴の回答から無関心または消極的な意思が見られない
4. 抽象化と統合:抽出した「重要」なスロット情報をもとに、それらを最もよく表す抽象的なキーワード（例：「キャリア・職場」「価値観」「趣味」）にまとめてペルソナ属性とします。
  - 既存の属性名や分類に明らかな誤りがある場合や、より適切な属性名にまとめられる場合は、統合・修正してください。（例：「職業観」と「現在のキャリア」→「キャリア・職場」に統合）
5. 状態の更新ルール：
  - 既存の推定ペルソナで「未」の属性について、重要な情報が得られた場合は「済」に更新します。
  - 既存の推定ペルソナにない新しい「重要」な情報が得られた場合は、新しいペルソナ属性を「済」として追加します。
  - 既存の状態が「未」であれ「済」であれ、今回の対話で尋ねた結果、スロットの値がルール2.で定義した「重要でない情報」（「情報なし」「わかりません」等）になった属性については、推定ペルソナから削除します.
  - 上記（「済」に更新、追加、削除）のいずれにも該当しない属性（例：既存が「未」で今回も尋ねられなかった属性、既存が「済」で今回は言及がなかった属性）は、既存の状態を維持します。
6. 出力はペルソナ属性名と状態（未/済）のみとします。
7. 複数の新しい属性が同時に見つかった場合でも、それぞれが「重要」の基準を満たすならば、全て追加してください。ただし、属性名が重複したり、内容が包含関係にある場合は、適切に統合してください。

# スロットの説明
スロットはインタビュー対象者のペルソナ情報を整理・記録するために使用します。
各スロットはスロット名、値の2つの要素で構成されます。
- スロット名: どのような情報が記録されているかを示すスロットの名前です。
- 値: インタビューで話された情報を端的にまとめて記録します。nullの場合は、まだ情報が聞き出されていない状態です。
- 例: {{"現在のキャリア": "大学病院の内科で勤務", "ストレス解消方法": null}}

# 出力指示
コメントや説明は不要です。
推定したペルソナ属性名と状態（未/済）を1行ずつ出力してください。
ペルソナ属性名は抽象的なキーワードを使い、かつ重複しないようにしてください。
インタビューの対話履歴と既存スロットを見て、既に十分な情報が得られたペルソナ属性は、その属性の後に「済」と記載してください。
インタビューの対話履歴と既存スロットを見て、新たに明らかになったペルソナ属性があれば「済」として積極的に推定ペルソナに追加してください。ただし、重要でないと判断した場合は追加しないでください。
インタビューでまだ情報が得られていないペルソナ属性には「未」と記載してください。

# 出力形式
[推定ペルソナ属性名]: 済
[推定ペルソナ属性名]: 未
[推定ペルソナ属性名]: 済

# 必ず確認するべきことの例
「株式投資」について「わかりません」のような回答 → 推定ペルソナに含めない
「思い出・エピソード」について有効な回答 → 思い出・エピソード: 済 を追加
既存の推定ペルソナで 株式投資: 未 があっても「わかりません」や情報が無い場合、そのような推定ペルソナは削除
既存で 趣味: 未 があって、趣味についての有効な回答があれば 趣味: 済 に更新
既存で「職業観」と「現在のキャリア」が両方ある場合 → 「キャリア・職場」に統合
既存の属性名や分類に明らかな誤りがある場合 → 適切な属性名に修正
属性名が具体的すぎる場合 → より抽象的な属性名に統合、または削除
重複や包含している推定ペルソナがある場合 → より抽象的なペルソナに包含
「学生時代の思い出」と「思い出・エピソード」を「思い出・エピソード」にまとめる

# 対話履歴
インタビュアー: 最近、特に昨日のことですが、何か悲しい出来事がありましたか？もしお話しできる範囲で教えていただければと思います。
インタビュー対象者: 特に悲しいことはありませんでした
インタビュアー: 海外旅行に行かれる際、どのような楽しみ方をされることが多いですか？例えば、観光地巡りや食べ物を楽しむなど、教えていただけますか？
インタビュー対象者: めったに海外旅行には行かないです
インタビュアー: 好きな芸能人や、特に注目している方がいれば教えていただけますか？
インタビュー対象者: あまり芸能人には詳しくないです
インタビュアー: これまでの人生で特に尊敬している人や、その理由について教えていただけますか？
インタビュー対象者: 今すぐには思いつかないですね
インタビュアー: 普段の生活の中で、特に楽しいと感じることや、趣味として楽しんでいることがあれば教えていただけますか？
インタビュー対象者: 趣味はファッション雑誌を読むことと、SNSでコーディネートを研究することです
インタビュアー: これまでのご経歴やお仕事について、どのようなことをされてきたのか教えていただけますか？
インタビュー対象者: アパレル系の専門学校を卒業した後、今のブランドにアルバイトとして入社し4年目です
インタビュアー: ご家族についてお伺いしてもよろしいですか？例えば、どのようなご関係やエピソードがあるのか教えていただけると嬉しいです。
インタビュー対象者: 両親と妹がいて、自分も実家暮らしです
インタビュアー: 食べ物の中で、特に苦手なものや嫌いなものがあれば教えていただけますか？
インタビュー対象者: 食べ物の好き嫌いは特にないです
インタビュアー: ファッションに対して、どのような価値観やこだわりをお持ちですか？例えば、選ぶ際に重視しているポイントなどがあれば教えてください。
インタビュー対象者: 雑誌などに乗っているおしゃれな服装を真似してしまいます
インタビュアー: ファッションに関して、将来的にどのような展望や目標をお持ちですか？例えば、挑戦してみたいことや夢などがあれば教えてください。
インタビュー対象者: 将来的には店長や商品を企画するプレス・MDの仕事にも挑戦したいです
インタビュアー: 今後のキャリアについて、昇進や転職などで考えていることや目標があれば教えていただけますか？
インタビュー対象者: まずは今の職場で正社員登用制度に挑戦することを目指しています
インタビュアー: 普段、作るのが得意な料理や、よく作る料理があれば教えていただけますか？
インタビュー対象者: 料理はあまりしないですね
インタビュアー: 資産運用やお金の管理について、どのような考え方をお持ちですか？例えば、貯金や投資などについて意識していることがあれば教えてください。
インタビュー対象者: 意識するべきなのですが、あまり考えていないですね
インタビュアー: ペットを飼っていらっしゃいますか？もし飼っている場合、どんな動物か教えていただけますか？
インタビュー対象者: 飼いたいとは思っているのですが、飼っていないです
インタビュアー: 普段、SNSをどのように活用されていますか？例えば、どんな投稿を見たり、どんな情報を発信したりしているのか教えていただけますか？
インタビュー対象者: SNSで他の人のコーディネートを研究しています
インタビュアー: 普段、スポーツジムに通ったり運動をする習慣はありますか？もしあれば、どのような運動をされているのか教えてください。
インタビュー対象者: 運動は滅多にしないです
インタビュアー: 朝起きた後、普段どのようなルーティンや習慣を持っていますか？例えば、最初にすることや気をつけていることがあれば教えてください。
インタビュー対象者: 特に決まったルーティンは持っていないです
インタビュアー: 普段、何かあったときに通われる病院やクリニックはありますか？例えば、風邪をひいたときなどに行く場所があれば教えてください。
インタビュー対象者: 特に決まった病院はないですね
インタビュアー: 普段の生活で健康を意識していることや、気をつけていることがあれば教えていただけますか？
インタビュー対象者: 意識するべきだとは思いますが、あまり気を付けられていません
インタビュアー: 普段、ご自身の性格についてどのように感じていますか？例えば、周りの方から言われることや、自分で思う特徴などがあれば教えてください。
インタビュー対象者: 明るく、頼られると断れないので責任感が強い性格だと思っています
インタビュアー: 普段、どのような音楽を聴くことが多いですか？好きなジャンルやアーティストがいれば教えてください。
インタビュー対象者: 決まったジャンルはないです、流行りの曲を聴くくらいですね
インタビュアー: 普段、語学を学ぶ際に意識していることや、工夫しているポイントがあれば教えていただけますか？
インタビュー対象者: 語学を学ぶ機会がめったにないです
インタビュアー: 将来的にどのような生活スタイルを送りたいと考えていますか？例えば、住む場所や日々の過ごし方など、イメージがあれば教えてください。
インタビュー対象者: 一人暮らしをしたいと思っていますが、収入のことを考えるとなかなか踏み切れないです
インタビュアー: 普段、大切にしている言葉や座右の銘があれば教えていただけますか？
インタビュー対象者: 今すぐには思いつかないです
インタビュアー: それでは、インタビューを終了します。ありがとうございました。
