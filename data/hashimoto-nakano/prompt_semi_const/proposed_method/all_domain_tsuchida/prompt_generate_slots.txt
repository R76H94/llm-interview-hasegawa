# タスク
あなたはインタビューの対話履歴・既存のスロット・推定ペルソナを分析し、直前までのインタビュー内容を踏まえてさらに深掘るための「追加で聞くべき観点」を見つけ、それを表す新しいスロットを1つだけ考え、生成してください。

# 前提
- スロットはインタビュー対象者の情報を整理・記録するために使用します。
- 生成するスロットは、すでに話題に出ている内容や既存スロットの情報を踏まえて、「今このタイミングでさらに深堀りして聞くと効果的な観点」をもとに生成してください。
- 生成するスロットのスロット名は「抽象度の高いキーワード」（例：「キャリア・職場」「悩みやストレス」「趣味」「家族」など）である必要があります。
- 既存のスロットにすでに含まれている観点と意味的に重複するスロットは生成しないでください。

# 新しいスロットの作成方法
1. 対話履歴（特に直前の話題）を分析し、インタビュー対象者が何について語ったかを把握してください。
2. 対話の内容から、次に掘り下げるとより深い人物理解につながりそうな観点を1つ見つけてください。
   - 例：話題が「仕事」に関するものであれば→「キャリアの方向性」や「仕事観」など。
3. 既存のスロットと推定ペルソナを確認し、すでに網羅されている観点や重複する話題は除外してください。
4. 抽出した観点を抽象的なスロット名として表現してください。
   - スロット名は短く簡潔に。具体的な文章ではなく、抽象的なカテゴリとする（例：「価値観」「将来の展望」「人間関係」など）。
5. 既存スロット・推定ペルソナと意味的に重複していないことを確認してください。
6. スロットの値は必ずnullにしてください。

# スロットの説明
スロットはインタビュー対象者のペルソナ情報を整理・記録するために使用します。
各スロットはスロット名、値の2つの要素で構成されます。
- スロット名: どのような情報が記録されているかを示すスロットの名前です。
- 値: 新しく作成するスロットは常に値をnullに設定してください。
例: {{"スロット名": null}}

# 推定ペルソナ
推定ペルソナは、これまでのインタビューですでに聞き出された／想定される観点を抽象化したものです。
- 「済」はすでにインタビューで聞き出された観点
- 「未」はまだ聞かれていないが、今後対象者が持っている可能性のある観点
新しいスロットは、この推定ペルソナに含まれていない観点（＝まだ想定されていない新しい切り口）を発見し、それを抽象的なスロット名として生成してください。
すでに推定ペルソナに存在する観点（済・未問わず）と同じ、または類似するスロットは生成しないでください。
{estimate_persona}

新しいスロットを1つのみ生成してください。
既存スロットは出力に含めないで、新たに作成したスロットのみを出力してください。
