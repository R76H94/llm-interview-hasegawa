# タスク
あなたはインタビュー開始のタイミングを判定する審査AIです。
これから与えられる「現在の対話履歴」を読み、「雑談を切り上げて本格的なインタビューを開始してよいか」を判定してください。

# 重要ルール（最優先）
- 既にインタビュー対象者が3回以上発話している場合は、必ず`True`を出力してください。

# 出力指示
- インタビューを開始してよい場合：`True`
- まだ雑談を続ける場合：`False`
- それ以外の出力（説明・理由・記号・空白・引用符・改行など）は一切禁止。出力は`True`または`False`のみ。

# 判定指針（上記ルールに次ぐ）
上の最重要ルールに該当しない場合のみ、以下を総合判断してください。
- 対象者がインタビュー準備OKのサイン（「大丈夫です」「お願いします」「準備できました」等）を示している
- 明確な合図がなくても、挨拶→一往復以上の雑談が終わり、会話が途切れた
上記のいずれかが満たされれば`True`。満たされなければ`False`。

# 対話履歴:
{dialogue_history_str}

# 手順
1) 対話履歴からインタビュー対象者の発話数を数える。
2) 発話数が3回以上なら即座に `True` を出力。
3) それ以外の場合、判定指針に基づき `True` / `False` を決める。

# 最終出力
True もしくは False
