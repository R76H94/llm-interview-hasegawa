# タスク
あなたはインタビューの対話履歴を分析し、インタビュー対象者のペルソナ情報をスロットに追加します。
スロットにペルソナ情報を追加する際は、インタビュー対象者が回答で使用した言葉や用語を反映させます。
既に情報が入力されているスロットに情報を追加することもできます。

# スロットの説明
スロットを用いてインタビュー対象者のペルソナ情報を整理し、記録します。
各スロットはスロット名、value の2つの要素で構成されます。
- スロット名: ペルソナ情報を記録するスロットの名前です。
- value: スロットに記録するペルソナ情報です。
- 例: {{"現在のキャリア": {{"value": "大学病院の内科で勤務"}},"悩みや不満点": {{"value": "人間関係がうまくいかない"}}}}

# スロットへの情報追加方法
1. 既存のスロットと対話履歴を分析し、インタビュー対象者の回答内容のうち、スロットに記録されていない情報を特定する。
2. その情報を既存のスロットのうち、最も最適と思われるスロットに追加する。既存のスロットに追加できない場合は、新たにスロットを作成せず、最も近い意味を持つスロットに追加する。
3. スロットに追加されていない情報は、昔の対話であっても該当するスロットに追加する。

# 出力指示
必ず JSON 形式で出力してください。
元のスロットをすべて出力し、いかなるスロットも削除しないでください。
新規のスロットは絶対に作成しないでください。
value のみに情報を追加してください。

# 出力例
{{
  "スロット名1": {{ // スロット名
    "value": "ここに情報を入力、追加" // スロットに情報を入力
  }},
  "スロット名2": {{ // スロット名
    "value": null // 埋められない場合はnullのままにしてください
  }}
}}
