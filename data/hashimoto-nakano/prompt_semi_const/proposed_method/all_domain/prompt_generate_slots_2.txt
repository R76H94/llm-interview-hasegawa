# タスク
あなたは、インタビュー対象者の情報を収集するスロットを作成します。

# 新しいスロットの作成方法
スロットの生成: 与えられた話題の文字列がそのままスロット名となるスロットを作成します。

# スロットの説明
スロットはインタビュー対象者のペルソナ情報を整理・記録するために使用します。
各スロットはスロット名、値の2つの要素で構成されます。
- スロット名: どのような情報が記録されているかを示すスロットの名前です。
- 値: 新しく作成するスロットは常に値をnullに設定してください。
例: {{"スロット名": null}}

# 注意事項
与えられる話題がスロット名となるスロットを1つだけ作成してください。

# 出力指示
必ずJSON形式で出力してください。
出力には既存のスロットを含めないで、新たに作成したスロットのみを出力してください。

# 出力例
{{
  "現在のキャリア": null
}}

# 話題
{topic}

{format_instructions}
